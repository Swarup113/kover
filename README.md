# Kover — Academic Cover Page Generator

## Live Demo
https://swarup113.github.io/kover/

---

## Overview
Kover is a lightweight academic cover page generator designed to eliminate formatting hassle. Instead of manually structuring documents in Word, users input their details once and get a clean, properly formatted cover page instantly.

It focuses on structure, consistency, and speed—ensuring every cover page follows a professional academic layout.

---

## Features

### Structured Input System
Multi-step form divided into logical sections:
- Institution Information  
- Document Details  
- Course Information  
- Student Information (supports up to 2 students)  
- Submission Details  
- University Logo upload  

This prevents formatting mistakes and enforces a consistent layout.

---

### Real-Time Preview
- Instant visual rendering of the cover page  
- WYSIWYG output (what you see is what you download)  
- Eliminates guesswork before exporting  

---

### Word Document Export
- Download as fully editable **.doc file**  
- Pre-formatted and ready for submission  
- Maintains proper academic styling  

---

### Academic Formatting Standards
- Default font: **Times New Roman**  
- Hierarchical typography:
  - University Name → larger emphasis  
  - Document Title → highlighted  
  - Body content → standard 12pt  
- Clean alignment and structured spacing  

---

### Multi-Student Support
- Displays up to two students side-by-side  
- Useful for group assignments  

---

### Minimal & Focused UI
- No unnecessary complexity  
- Direct workflow: Input → Preview → Download  

---

## Built With
- HTML5  
- CSS3  
- Vanilla JavaScript  

---

## Why Kover
Most students waste time adjusting spacing, alignment, and formatting in Word. Kover removes that friction by standardizing the process and generating a submission-ready cover page in seconds.
