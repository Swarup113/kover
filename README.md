# Kover – Academic Cover Page Generator

### *A lightweight academic cover page generator that creates clean, properly formatted cover pages from structured input*

Stop wrestling with Word formatting. Kover provides a structured multi‑step form, real‑time preview, and one‑click export to a fully editable .doc file. Perfect for students, researchers, and academics who need submission‑ready cover pages in seconds.

## Overview

| Aspect | Description |
|--------|-------------|
| Purpose | Eliminate manual formatting hassle in Word. Generate a professional academic cover page from structured input. |
| Workflow | Input details → Real‑time preview → Download .doc file |
| Target Users | Students, researchers, academics preparing assignments, theses, or reports. |
| Key Benefit | Consistency, speed, and proper academic layout without trial‑and‑error formatting. |

---

## Features

| Feature | Description |
|---------|-------------|
| Structured Input System | Multi‑step form with logical sections: Institution Information, Document Details, Course Information, Student Information (up to 2 students), Submission Details, University Logo upload. Prevents formatting mistakes. |
| Real‑Time Preview | Instant visual rendering of the cover page. WYSIWYG output – what you see is what you download. Eliminates guesswork before export. |
| Word Document Export | Download as a fully editable **.doc file**. Pre‑formatted and ready for submission. Maintains proper academic styling. |
| Academic Formatting Standards | Default font: Times New Roman. Hierarchical typography (University Name larger, Document Title highlighted, body content standard 12pt). Clean alignment and structured spacing. |
| Multi‑Student Support | Displays up to two students side‑by‑side. Ideal for group assignments. |
| Minimal & Focused UI | No unnecessary complexity. Direct workflow: Input → Preview → Download. |

---

## How to Use

| Step | Action |
|------|--------|
| 1 | Open Kover in your browser. |
| 2 | Fill in the multi‑step form – Institution, Document Title, Course, Student names, Submission date, and optional University Logo. |
| 3 | Watch the real‑time preview update instantly as you type. |
| 4 | Adjust any field until the preview looks exactly as you want. |
| 5 | Click the **Download** button to export a ready‑to‑submit .doc file. |
| 6 | Open the downloaded file in Microsoft Word or any word processor – it’s fully editable. |

> No registration, no data uploads, no server processing – everything runs locally in your browser.
